# Видеоконвертер

Веб-приложение для конвертации видеофайлов в различные форматы на основе Flask и FFmpeg.

## Возможности

- Конвертация видео в форматы: MP4, WEBM, AVI, MKV
- Поддержка входных форматов: MP4, AVI, MOV, MKV, WEBM
- Отображение информации о видео (разрешение, длительность, кодеки)
- Асинхронная обработка файлов
- Современный пользовательский интерфейс на Bootstrap 5

## Требования

- Python 3.8+
- FFmpeg (должен быть установлен и доступен в PATH)

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/converter.git
cd converter
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Убедитесь что FFmpeg установлен:
```bash
ffmpeg -version
```

## Запуск

```bash
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## Структура проекта

```
converter/
├── app.py                  # Основной файл приложения
├── requirements.txt        # Зависимости Python
├── templates/              # HTML шаблоны
│   ├── base.html
│   ├── index.html
│   ├── processing.html
│   ├── result.html
│   └── error.html
├── uploads/                # Временная папка для загруженных файлов
└── converted/              # Временная папка для сконвертированных файлов
```

## Использование

1. Откройте http://localhost:5000 в браузере
2. Выберите видеофайл для конвертации
3. Выберите целевой формат
4. Нажмите "Загрузить и конвертировать"
5. Дождитесь завершения обработки
6. Скачайте готовый файл

## Ограничения

- Максимальный размер файла: 500 МБ
- Максимальное время конвертации: 10 минут

## Технологии

- **Backend**: Flask
- **Конвертация**: FFmpeg
- **Frontend**: Bootstrap 5, Font Awesome
- **Шаблоны**: Jinja2
